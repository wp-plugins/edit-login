== Description ==

Edit Login plugin allows you to edit the wordpress default login page: customize easily the login page background, the logo and its link

== Installation ==

1. Upload 'editLogin.php' and 'sample.jpg' to the '/wp-content/plugins/editLogin/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
