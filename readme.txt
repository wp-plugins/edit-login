== Description ==

Edit Login plugin allows you to edit the wordpress default login page: customize easily the login page background and font, the logo and its link

== Installation ==

1. Upload 'editLogin.php' and 'sample.jpg' to the '/wp-content/plugins/editLogin/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= Edit Login plugin is easy to use? =
Yes, after the installation you have to set only three parameters
= Can i use only font from Google Fonts? =
Yes, you can use only fonts from Google Fonts or leave the default theme font
